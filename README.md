# My Portfolio Landing Page

## 1. Overview of Portfolio Purpose
This portfolio was built as part of the Web Authoring module. It introduces who I am as a student developer and provides navigation to project documentation pages. The goal is to demonstrate HTML and CSS fundamentals while building a real personal portfolio.

## 2. Description of Page Contents
- **index.html** — Landing page with hero section, about summary, skills list, and contact section
- **about.html** — A page describing my background, learning journey, and goals, with a link to my GitHub profile
- **contact.html** — A page where visitors can contact me and find links to my profiles on the web
- **styles.css** — External stylesheet used across all pages for consistent design

## 3. Technologies Used
- HTML5
- CSS3 (external stylesheet)
- Git & GitHub for version control
- GitHub Pages for hosting
